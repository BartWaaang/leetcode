# [Excel Sheet Column Number](https://leetcode.com/problems/excel-sheet-column-number/)

---

## Description

Given a column title as appear in an Excel sheet, return its corresponding column number.

For example:

    A -> 1

    B -> 2

    C -> 3

    ...

    Z -> 26

    AA -> 27

    AB -> 28

---

问题实际相当于将27进制转换成10进制
